## Code example
[Get the Middle Character](https://www.codewars.com/kata/56747fd5cb988479af000028): You are going to be given a word. Your job is to return the middle character of the word. If the word's length is odd, return the middle character. If the word's length is even, return the middle 2 characters.

```
function getMiddle(s) {
  let middle = s.length / 2;

  return (s.length % 2) ? s[Math.floor(middle)] : s.slice(middle - 1, middle + 1);
}
```
